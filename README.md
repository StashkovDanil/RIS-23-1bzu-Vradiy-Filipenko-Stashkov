# Создание программы для кормушки для животного с настройкой времени и количества грамм выдаваемого корма
Группа: __РИС-23-1бзу.__

Команда: __Врадий Никита, Филипенко Никита, Сташков Данил.__

## Запланированный функционал:
1. Кнопка кормежки - незамедлительно выдает порцию заданного размера. 
2. Выдача корма по расписанию - настройка расписания автоматической выдачи. 
3. Уведомления - о выдачах по расписанию, о заканчивающемся корме. 
4. Мониторинг - журнал кормежек, количество оставшегося корма.
